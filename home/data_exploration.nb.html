Filling missing categorical data

The missing data maybe filled in the following ways: - Gender, Dependents, Self_Employed, Credit_History: use mode to impute - Married: small enough to try profiling and using mode - LoanAmount and Loan_Amount_Term: may need to discard since it is what we are predicting

# get mode function definition
getmode <- function(v) {
   uniqv <- unique(v)
   uniqv[which.max(tabulate(match(v, uniqv)))]
}
